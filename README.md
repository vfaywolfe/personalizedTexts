Generic texts are web-viewable from: 
* vfaywolfe.github.io/personalizedTexts/books/[bookname]/Chapter[X]/html/chapter\_[X]\_lesson\_[Y].html

Where [X] is the chapter number and [Y] is the lesson number. For example:
* vfaywolfe.github.io/personalizedTexts/books/CSC106/Chapter2/html/chapter\_2\_lesson_1.html
