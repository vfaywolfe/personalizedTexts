What is Python?

Python is a text-based programming language. Instead of dragging and dropping blocks like in Code.org or other visual environments, you write actual code commands that the computer reads and executes. Don't worry‚Äîit's much more straightforward than it might look!

Python got its name from the British comedy series "Monty Python," and its creator wanted the language to be as entertaining and easy to use as a comedy show. Python is used everywhere:

  • Data Science: Analyzing massive datasets
  • Machine Learning: Training AI models
  • Web Development: Building websites and applications
  • Automation: Creating scripts to do repetitive tasks

The Print Statement: Your First Command

The simplest way to start programming in Python is with the print() statement. It tells Python to display something on the screen.

üìö Key Terms

Print Statement
A command that displays output on the screen. In Python, you write print() followed by what you want to display in parentheses.
String
Text data in Python, always enclosed in quotation marks (either single '' or double ""). Example: "Hello, World!"
Expression
A combination of values and operations that Python can evaluate to produce a result. Example: 2025 - 1990

Simple Examples

Step 2: Output:

Example 9.1.3: Python as a Calculator

Step 1: Write an expression that Python will evaluate
print(2025 - 1990)
Python evaluates (solves) the math problem and prints the result.
Step 2: Output:
35

Understanding Data Types

‚è±Ô∏è 8 minutes | Lesson 2 of 9

What are Data Types?

Every piece of information your program uses has a type. Think of data types like different ingredients in a recipe‚Äîyou need to know whether you're working with flour (a solid), milk (a liquid), or an egg (something different altogether). Python needs to know the same thing about your data.

üìö Data Types in Python

String (str)
Text data enclosed in quotation marks. Examples: "Alice", 'Hello', "123 Main Street". Even though the last example contains numbers, it's a string because it's in quotes.
Integer (int)
A whole number with no decimal point. Can be positive or negative. Examples: 42, -10, 0, 2024
Float (float)
A number with a decimal point. Examples: 3.14, 2.5, -0.5, 98.6

Why Data Types Matter

Python treats different data types differently. Watch what happens:

Example 9.2.1: Strings vs. Integers

With strings:
print("5" + "3")
Python combines the text: outputs 53
With integers:
print(5 + 3)
Python adds the numbers: outputs 8
‚ö†Ô∏è

Common Mistake: Don't mix quotes for numbers! print("5" + 3) will cause an error because Python can't add a string and a number‚Äîthey're different types.

Working with Different Data Types

Example 9.2.2: Floats for Precise Calculations

Calculating an average with floats:
print((90.5 + 87.3 + 92.1) / 3)
Using floats allows for decimal precision. Output: 89.96666666666667
‚ÑπÔ∏è

Fun Fact: You can convert between data types! Python provides functions like int(), str(), and float() to convert data from one type to another. We'll use these soon!

Creating and Using Variables

‚è±Ô∏è 10 minutes | Lesson 3 of 9
LO9.3.1:

Create variables to store data and use them in Python programs to make code more flexible and reusable.

What is a Variable?

A variable is like a labeled box where you can store information. Instead of writing the same value over and over, you store it once in a variable and then use the variable name whenever you need it.

Think of it like saving a contact in your phone. Instead of typing out someone's full phone number every time you want to call them, you just tap their name. The name is like a variable‚Äîit represents the phone number stored underneath.

üìö Key Terms

Variable
A named storage location that holds a value. You create a variable by writing: variable_name = value
Assignment
The process of giving a variable a value. The = sign is called the assignment operator.
Variable Name
The label you give to your variable. Should be descriptive. Examples: age, name, score, birth_year

Creating Variables

The syntax (format) is simple:

variable_name = value

Example 9.3.1: Creating and Using Variables

Step 1: Create a variable to store a name
name = "Alice"
The variable name now holds the string "Alice"
Step 2: Create a variable to store an age
age = 18
The variable age holds the integer 18
Step 3: Use variables in print statements
print(name) print(age)
When you use the variable name without quotes, Python prints the value stored in the variable.
Output:
Alice 18

Example 9.3.2: Using Variables in Expressions

Step 1: Create variables
birth_year = 2006 current_year = 2024
Step 2: Use variables in a calculation
age = current_year - birth_year print(age)
Instead of calculating manually, let Python do the math using your variables.
Output:
18

Variable Naming Rules

Python has rules about variable names. Following these rules will save you from errors:

  • Variable names must start with a letter or underscore (_)
  • Variable names can contain letters, numbers, and underscores
  • Variable names cannot contain spaces (use underscores instead: birth_year, not birth year)
  • Variable names are case-sensitive: Name and name are different variables
  • Use descriptive names: student_score is better than ss
üí°

Best Practice: Use lowercase with underscores for variable names (called "snake_case"). For example: favorite_color, total_score, is_active

Combining Variables with Strings

Creating Meaningful Output

Printing just the variable value isn't very helpful: print(age) outputs 18. But what does 18 represent? To make your program more informative, you need to combine variables with explanatory text.

üìö Key Terms

Concatenation
Combining strings together using the + operator. Example: "Hello" + " " + "World" becomes "Hello World"
Type Conversion
Converting a value from one data type to another. Use str() to convert numbers to strings, so they can be concatenated with text.

The Challenge: Numbers Aren't Strings

Remember from Lesson 9.2 that you can't concatenate strings with numbers directly. This won't work:

Python says: "I can't add a string and a number!" You need to convert the number to a string first using the str() function.

Solution: Use str() to Convert Numbers

Example 9.4.1: Converting Numbers to Strings

Step 1: Create variables

Step 2: Convert numbers to strings and concatenate
print("My name is " + name) print("I am " + str(age) + " years old") print("My GPA is " + str(gpa))
The str() function converts the integer and float to strings so they can be combined with the text.
Output:
My name is Bob I am 25 years old My GPA is 3.85

Step 2: Create a calculated variable

Output:
Hello, my name is Sarah I am 19 years old I was born in 2005

Remember: You only need to use str() when concatenating with +. If you're just printing a number by itself, no conversion is needed!

Comments and Debugging Your Code

‚è±Ô∏è 12 minutes | Lesson 5 of 9
LO9.5.1:

Write comments to document code and use error messages and stack traces to identify and fix bugs in Python programs.

Why Comments Matter

Imagine reading a recipe with no explanations‚Äîjust a list of steps. Comments are like notes in a recipe that explain what you're doing and why. They're especially important when:

  • You come back to old code months later and need to understand it
  • Someone else reads your code
  • You're debugging and need to understand what each section does

üìö Key Terms

Comment
A note in your code that Python ignores. Starts with #. Example: # This is a comment
Bug
An error in your code that prevents it from running correctly.
Stack Trace
The error message Python displays when something goes wrong. It tells you what the error is and where it happened.

Writing Comments

Example 9.5.1: Using Comments to Explain Code

Code with helpful comments:
# Store student information name = "Michael" age = 20 # Calculate birth year current_year = 2024 birth_year = current_year - age # Display the information print("Student: " + name) print("Birth year: " + str(birth_year))
Comments (lines starting with #) explain what each section does. Python ignores them completely.
üí°

Comment Best Practices: Use comments to explain WHY you did something, not WHAT the code does. The code itself shows what it does. "# Calculate the person's age" is better than "# age = 2024 - birth_year" (that's just repeating the code).

Understanding Errors and Stack Traces

When Python encounters an error, it gives you helpful information. Let's learn how to read these error messages:

Example 9.5.2: Reading a Stack Trace

Buggy code:
name = "Jordan" age = 21 print("My name is " + name + age)
Error message you'd see:
TypeError: can only concatenate str (not "int") to str at line 3
Python says: "You're trying to add a string and an integer. That doesn't work!" The error tells you the problem AND where it happened (line 3).
Fixed code:
name = " Jordan" age = 21 print("My name is " + name + " and I am " + str(age))
Convert the integer to a string using str(age). Now all parts are strings and can be concatenated.

Common Python Errors

Here are errors you might encounter and what they mean:

‚ö†Ô∏è

SyntaxError: You made a typo in your code. Common causes: missing quotes, missing parentheses, misspelled keywords. Python can't even try to run the program.

‚ö†Ô∏è

NameError: You used a variable name that doesn't exist. Either you misspelled it or forgot to create it. Example: print(names) when you created name (without the 's').

‚ö†Ô∏è

TypeError: You tried to do an operation with the wrong types. Example: adding a string and an integer, or trying to do math on text.

Creating and Using Lists

‚è±Ô∏è 11 minutes | Lesson 7 of 9

What is a List?

A list is like a container that holds multiple pieces of information. Instead of creating separate variables for each item, you can store them all together in one list.

Think of a shopping list: instead of having separate sticky notes for "milk," "eggs," and "bread," you write them all on one piece of paper. Similarly, in Python, a list lets you organize related data in one place.

üìö Key Terms

List
A collection of values in a specific order, enclosed in square brackets. Example: ["apple", "banana", "orange"]
Index
The position of an item in a list, starting from 0. The first item is at index 0, the second at index 1, etc.
Element
An individual item in a list.

Creating a List

Example 9.7.1: Creating Lists

Step 1: Create a list of strings
fruits = ["apple", "banana", "orange", "grape"]
Square brackets [ ] contain the list. Items are separated by commas.
Step 2: Create a list of numbers
scores = [95, 87, 92, 88, 91]
Lists can contain any data type‚Äîstrings, integers, floats, or even a mix!
Step 3: Create a list with mixed types
student_info = ["Alice", 20, 3.85]
This list contains a string, an integer, and a float all together.

Accessing Items in a List (Indexing)

To get a specific item from a list, you use its index (position). Remember: Python starts counting at 0.

Example 9.7.2: Using Indexes

Step 1: Create a list
courses = ["Python", "Web Design", "Data Science", "AI"]
The list has 4 items at indexes 0, 1, 2, and 3.
Step 2: Access items by index
print(courses[0]) # First item print(courses[1]) # Second item print(courses[3]) # Fourth item
Use square brackets with the index number to get an item.
Output:
Python Web Design AI
‚ö†Ô∏è

Off-by-One Error: A common mistake! If your list has 4 items, the last index is 3, not 4. Using courses[4] would cause an IndexError.

üí°

Negative Indexing: Python also allows negative indexes! list[-1] gets the last item, list[-2] gets the second-to-last, and so on.

For Loops: Repeating Actions

‚è±Ô∏è 12 minutes | Lesson 8 of 9
LO9.8.1:

Write for loops to iterate through lists and automate repetitive tasks in Python.

What is a Loop?

Imagine you have a list of 100 students and want to print each name. Would you write 100 separate print statements? No! That's what loops are for. A loop repeats code automatically, once for each item in a list.

üìö Key Terms

For Loop
A loop that repeats code once for each item in a list. Syntax: for item in list:
Iteration
One complete cycle through the loop. If you loop over a 5-item list, there are 5 iterations.
Loop Variable
The variable that holds the current item. Example: in for fruit in fruits:, fruit is the loop variable.

For Loop Syntax

for item in list: # Code here runs for each item print(item)

Important: Notice the colon (:) at the end of the for statement and the indentation (spaces) on the next line. This tells Python that the indented code belongs to the loop.

Example 9.8.1: Simple For Loop

Step 1: Create a list
colors = ["red", "blue", "green", "yellow"]
Step 2: Write a for loop to print each color
for color in colors: print(color)
For each item in the colors list, print it. The loop runs 4 times (once for each color).
Output:
red blue green yellow

Making Loops More Interesting

Example 9.8.2: Loops with String Concatenation

Step 1: Create a list of courses
courses = ["Python", "Web Design", "Data Science"]
Step 2: Loop and add formatting
print("My Courses:") for course in courses: print("- " + course)
Add text formatting to make the output look better.
Output:
My Courses: - Python - Web Design - Data Science
‚ö†Ô∏è

Indentation is Critical! Python uses indentation to determine which code belongs to the loop. If you forget to indent, you'll get an error or unexpected behavior.

üí°

Choose Meaningful Loop Variable Names: Use for student in students: instead of for s in students:. It makes your code easier to understand.

Creating and Using Lists

‚è±Ô∏è 11 minutes | Lesson 7 of 9
LO9.7.1:

Create lists to store multiple values and access individual items using indexing.

What is a List?

A list is like a container that holds multiple pieces of information. Instead of creating separate variables for each item, you can store them all together in one list.

Think of a shopping list: instead of having separate sticky notes for "milk," "eggs," and "bread," you write them all on one piece of paper. Similarly, in Python, a list lets you organize related data in one place.

üìö Key Terms

List
A collection of values in a specific order, enclosed in square brackets. Example: ["apple", "banana", "orange"]
Index
The position of an item in a list, starting from 0. The first item is at index 0, the second at index 1, etc.
Element
An individual item in a list.

Creating a List

Example 9.7.1: Creating Lists

Step 1: Create a list of strings
fruits = ["apple", "banana", "orange", "grape"]
Square brackets [ ] contain the list. Items are separated by commas.
Step 2: Create a list of numbers
scores = [95, 87, 92, 88, 91]
Lists can contain any data type‚Äîstrings, integers, floats, or even a mix!
Step 3: Create a list with mixed types
student_info = ["Alice", 20, 3.85]
This list contains a string, an integer, and a float all together.

Accessing Items in a List (Indexing)

To get a specific item from a list, you use its index (position). Remember: Python starts counting at 0.

Example 9.7.2: Using Indexes

Step 1: Create a list
courses = ["Python", "Web Design", "Data Science", "AI"]
The list has 4 items at indexes 0, 1, 2, and 3.
Step 2: Access items by index
print(courses[0]) # First item print(courses[1]) # Second item print(courses[3]) # Fourth item
Use square brackets with the index number to get an item.
Output:
Python Web Design AI
‚ö†Ô∏è

Off-by-One Error: A common mistake! If your list has 4 items, the last index is 3, not 4. Using courses[4] would cause an IndexError.

üí°

Negative Indexing: Python also allows negative indexes! list[-1] gets the last item, list[-2] gets the second-to-last, and so on.

For Loops: Repeating Actions

‚è±Ô∏è 12 minutes | Lesson 8 of 9
LO9.8.1:

Write for loops to iterate through lists and automate repetitive tasks in Python.

What is a Loop?

Imagine you have a list of 100 students and want to print each name. Would you write 100 separate print statements? No! That's what loops are for. A loop repeats code automatically, once for each item in a list.

üìö Key Terms

For Loop
A loop that repeats code once for each item in a list. Syntax: for item in list:
Iteration
One complete cycle through the loop. If you loop over a 5-item list, there are 5 iterations.
Loop Variable
The variable that holds the current item. Example: in for fruit in fruits:, fruit is the loop variable.

For Loop Syntax

for item in list: # Code here runs for each item print(item)

Important: Notice the colon (:) at the end of the for statement and the indentation (spaces) on the next line. This tells Python that the indented code belongs to the loop.

Example 9.8.1: Simple For Loop

Step 1: Create a list
colors = ["red", "blue", "green", "yellow"]
Step 2: Write a for loop to print each color
for color in colors: print(color)
For each item in the colors list, print it. The loop runs 4 times (once for each color).
Output:
red blue green yellow

Making Loops More Interesting

Example 9.8.2: Loops with String Concatenation

Step 1: Create a list of courses
courses = ["Python", "Web Design", "Data Science"]
Step 2: Loop and add formatting
print("My Courses:") for course in courses: print("- " + course)
Add text formatting to make the output look better.
Output:
My Courses: - Python - Web Design - Data Science
‚ö†Ô∏è

Indentation is Critical! Python uses indentation to determine which code belongs to the loop. If you forget to indent, you'll get an error or unexpected behavior.

üí°

Choose Meaningful Loop Variable Names: Use for student in students: instead of for s in students:. It makes your code easier to understand.
