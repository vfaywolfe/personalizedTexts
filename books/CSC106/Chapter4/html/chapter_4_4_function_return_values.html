üìñ Introduction

You've learned how to create functions and how to pass information into them using parameters. This makes functions flexible‚Äîthey can accept different inputs and perform operations on that data. But what happens when you need to get information back from a function?

Think about asking a friend to calculate something for you. You give them the numbers (parameters), they do the math in their head (the function's code), and then they tell you the answer. That answer is what we call a return value. Without return values, functions could only perform actions like drawing shapes or printing text. With return values, functions can calculate results and give them back to you to use however you need.

Imagine you're building a game and you need to calculate the distance between the player and an enemy. You could write the distance formula code every time you need it, or you could write a calculateDistance() function that takes two positions as parameters and returns the distance. Then you can use that returned value to decide if the enemy is close enough to attack, if a bullet hit its target, or if the player collected a coin.

In this lesson, you'll learn how to use the return statement to send values back from functions. You'll discover how to store these returned values in variables, use them in calculations, and even pass them to other functions. This completes the circle: parameters let data flow into functions, and return values let results flow out. Together, they make functions incredibly powerful!

üìö Key Concepts and Terms

Return Statement

A line of code using the return keyword that sends a value back from a function to the code that called it. The return statement also immediately exits the function.

Return Value

The value that a function sends back using a return statement. This value can be stored in a variable, used in expressions, displayed, or passed to other functions.

Function Exit

The moment when a function stops executing and control returns to the code that called it. This happens when a return statement is executed or when the function reaches its closing brace.

Expression

Any code that produces a value. Return values can be used in expressions, such as var total = add(5, 3) + 10;

Void Function

A function that doesn't return a value‚Äîit only performs actions. Drawing functions are often void functions.

üìù Content

What Are Return Values?

So far, most of your functions have performed actions‚Äîdrawing shapes, displaying text, or changing colors. These functions do their work and then finish. But many functions need to calculate something and give you the result.

Consider these built-in functions you've been using:

  • random(1, 100) ‚Äî returns a random number between 1 and 100
  • sqrt(25) ‚Äî returns 5 (the square root of 25)
  • abs(-7) ‚Äî returns 7 (the absolute value of -7)

These functions calculate values and return them. You can use these returned values:

var num = random(1, 100); // Store the returned value var root = sqrt(25); // Store the returned value text(abs(-7), 100, 100); // Use the returned value directly

Now you'll learn to create your own functions that return values!

The Return Statement

To make a function return a value, use the return keyword followed by the value you want to send back:

function functionName() { return value; }
Example 4.4.1: Simple Function with Return Value
Step 1: Define a function that returns a value
function add(num1, num2) { return num1 + num2; }

This function accepts two parameters, calculates their sum, and returns the result using the return keyword.

Step 2: Call the function and use the returned value
var sum = add(5, 3); // sum now contains 8 text(sum, 100, 100); // Displays "8" at position (100, 100)

When you call add(5, 3), the function calculates 5 + 3 and returns 8. That value is then stored in the variable sum.

Step 3: Understanding the flow
  1. The code calls add(5, 3)
  2. Inside the function, num1 is 5 and num2 is 3
  3. The function calculates 5 + 3, which equals 8
  4. The return statement sends 8 back to the caller
  5. The returned value (8) is stored in the variable sum
Think of return as the function's answer. When you ask the function to do something, the return value is its response.

Return Immediately Exits the Function

Here's something critical to understand: when a function executes a return statement, it immediately stops and exits. Any code written after the return will never execute!

Example 4.4.2: Return Stops Execution
function demonstrateReturn() { text("This will display", 10, 30); return 42; // Function exits here! text("This will NEVER display", 10, 60); // This line never runs } var result = demonstrateReturn(); text("Returned value: " + result, 10, 90);

What displays:

  • "This will display" at (10, 30)
  • "Returned value: 42" at (10, 90)
  • The second text() inside the function never runs!

The function exits immediately when it hits the return statement.

Any code after a return statement is unreachable and will never execute. This is important for debugging‚Äîif you have code that's not running, check if there's a return statement before it!

Using Return Values

Return values are incredibly versatile. You can:

  • Store them in variables
  • Use them directly in expressions
  • Pass them to other functions
  • Display them on the screen
  • Use them in conditional statements
Example 4.4.3: Multiple Ways to Use Return Values
function double(num) { return num * 2; } // Method 1: Store in a variable var result = double(5); // result = 10 text(result, 10, 30); // Method 2: Use directly in an expression var total = double(3) + double(4); // 6 + 8 = 14 text(total, 10, 60); // Method 3: Pass to another function text(double(7), 10, 90); // Displays "14" // Method 4: Use in a calculation var area = double(5) * double(3); // 10 * 6 = 60 text(area, 10, 120);

All of these are valid! Return values can be used anywhere you would use a regular value.

Creating Calculation Functions

Return values are perfect for functions that perform calculations. Let's create some useful mathematical functions:

Example 4.4.4: Basic Math Functions
function add(a, b) { return a + b; } function subtract(a, b) { return a - b; } function multiply(a, b) { return a * b; } function divide(a, b) { return a / b; } // Using these functions var x = 15; var y = 3; text("15 + 3 = " + add(x, y), 10, 30); // Displays: 18 text("15 - 3 = " + subtract(x, y), 10, 60); // Displays: 12 text("15 √ó 3 = " + multiply(x, y), 10, 90); // Displays: 45 text("15 √∑ 3 = " + divide(x, y), 10, 120); // Displays: 5
Example 4.4.5: More Advanced Calculations
function square(num) { return num * num; } function cube(num) { return num * num * num; } function average(a, b) { return (a + b) / 2; } function celsiusToFahrenheit(celsius) { return (celsius * 9/5) + 32; } // Using these functions text("5 squared = " + square(5), 10, 30); // 25 text("3 cubed = " + cube(3), 10, 60); // 27 text("Average of 10 and 20 = " + average(10, 20), 10, 90); // 15 text("25¬∞C = " + celsiusToFahrenheit(25) + "¬∞F", 10, 120); // 77¬∞F

These functions encapsulate formulas, making your code more readable and reusable!

When to Use Return Values: If your function calculates something, it should return that value. If your function only performs actions (like drawing or changing state), it doesn't need a return statement.

Return Values in Decision Making

Return values are especially useful in conditional logic. You can create functions that return true or false to help make decisions:

Example 4.4.6: Boolean Return Values
function isEven(num) { return num % 2 === 0; } function isPositive(num) { return num > 0; } function isBetween(value, min, max) { return value >= min && value <= max; } // Using boolean return values var number = 7; if (isEven(number)) { text(number + " is even", 10, 30); } else { text(number + " is odd", 10, 30); } if (isPositive(number)) { text(number + " is positive", 10, 60); } if (isBetween(number, 1, 10)) { text(number + " is between 1 and 10", 10, 90); }

These functions return boolean values (true or false) that can be used directly in if statements.

Question 4 (Coding):

Write a function called square that takes one parameter (num) and returns the square of that number (num √ó num).

Show Sample Answer
function square(num) { return num * num; }

This function multiplies the parameter by itself and returns the result. Example: square(5) returns 25.

Question 5 (Application):

Write a function called average that takes two parameters and returns their average (the sum divided by 2).

Show Sample Answer
function average(num1, num2) { return (num1 + num2) / 2; }

This function adds the two numbers and divides by 2. Example: average(10, 20) returns 15.

Question 6 (Analysis):

What's wrong with this function?

Issue: This function calculates total but doesn't return it. If you want to use the calculated value elsewhere, you need to add return total; at the end. Currently, the function only displays the value and doesn't make it available for use in other parts of the program.
