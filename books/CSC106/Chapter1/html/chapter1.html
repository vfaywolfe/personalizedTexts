1.1 Course Introduction

Welcome to CSC 106: The Joy of Programming! This course is your gateway into the exciting world of computer programming. Whether you're pursuing a degree in computer science, exploring programming out of curiosity, or building skills for your future career, this course will give you a solid foundation in fundamental programming concepts.

In CSC 106, you'll learn to think like a programmer—breaking down complex problems into manageable pieces, designing solutions before you code, and systematically debugging when things don't work as expected. You'll write programs using JavaScript and the P5.js library, creating everything from simple drawings to interactive animations and even complete games. Along the way, you'll learn how to work effectively with AI tools as tutoring assistants, building the critical thinking skills necessary to use these powerful technologies responsibly and effectively.

Course Learning Objectives

By the end of this course, you will be able to:

LO1.1: Write Fundamental Programs Independently
You will write simple programs using variables, functions, conditionals, loops, and arrays without external assistance. You will demonstrate mastery of basic programming constructs through individual coding exercises and assessments.
LO1.2: Design Programs Before Coding
You will analyze problems and create program designs using AI-assisted thought partnering, pseudocode, flowcharts, or other planning tools before implementation. You will apply structured problem-solving approaches to break down complex tasks into manageable components.
LO1.3: Debug and Test Programs Systematically
You will identify, troubleshoot, and correct syntax, logic, and runtime errors using systematic debugging methods. You will develop and execute test plans to verify program functionality and handle edge cases.
LO1.4: Utilize AI and External Resources Effectively
You will effectively and responsibly use AI and online resources to solve large programming challenges, understanding when to seek help and how to learn from assistance rather than simply copying solutions.
LO1.5: Develop a Comprehensive Final Project
You will design and implement a substantial program (such as a game or educational app) that integrates multiple programming concepts. You will manage the complete development process from initial design through final presentation and demonstration.

1.4 What's Next: Your Programming Journey Begins

🚀 Ready to Start Coding!

Now that we've gotten to know you and assessed your starting point, it's time to dive into the exciting world of programming! In Chapter 2, you'll start writing your first programs using JavaScript and the P5.js library.

What You'll Learn in Chapter 2:

  • Drawing with Code: You'll learn how to use JavaScript commands to draw shapes, lines, and colors on the screen. Think of it as giving instructions to a computer artist!
  • The Coordinate System: You'll discover how computers use a Cartesian coordinate system (X, Y positions) to place objects on the screen—kind of like a digital treasure map.
  • Your First Variables: You'll learn what variables are and how to use them to store information your programs need. Variables are like labeled boxes where you can keep data.
  • Functions and Parameters: You'll understand how functions work and how to pass information to them using parameters. This is how you'll tell the computer exactly what shapes to draw and where to put them.

What to Expect:

Chapter 2 is hands-on and interactive. You'll be writing code from the very first lesson, starting with simple commands and gradually building up to more complex programs. Don't worry if things seem confusing at first—programming is a skill that develops with practice, and you'll have plenty of opportunities to experiment and learn.

Tips for Success:

  • Experiment: Try changing numbers in your code to see what happens. This is one of the best ways to learn!
  • Make mistakes: Everyone makes errors in their code. Debugging (finding and fixing errors) is a normal part of programming.
  • Use the AI tutor: Remember, CSC 106 has an AI tutor built into CodeLab. Use it to ask questions and get explanations.
  • Practice regularly: Programming is like learning a musical instrument—regular practice is more effective than cramming.
  • Have fun: You'll be creating visual programs you can share with friends and family. Enjoy the creative process!
